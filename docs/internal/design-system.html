Design Tokens

Canonical token reference for spacing, typography, radius, elevation, motion, and state values used across Unimad.

Core Color Tokens

Brand

`brand-50` → `brand-900`

Primary action: `brand-600`, Hover: `brand-700`, Focus ring: `brand-500/20`

Neutral / Navy

`slate-50` → `slate-950`

Primary navy CTA: `slate-900`; dark surfaces: `slate-900` / `slate-950`

Spacing Scale (4px Base)

`1` = 4px
`2` = 8px
`3` = 12px
`4` = 16px
`5` = 20px
`6` = 24px
`8` = 32px
`10` = 40px
`12` = 48px
`16` = 64px

Preferred component spacing: 8, 12, 16, 20, 24.

Typography Tokens (Onest)

Display / Page Titles

`text-3xl font-semibold`

Section Heading

`text-xl font-medium`

Body

`text-sm font-normal|light leading-relaxed`

Meta / Micro

`text-xs` or `text-[10px] font-medium uppercase tracking-widest`

Radius, Border, Shadow Tokens

Radius

`rounded-md` (6px), `rounded-lg` (8px), `rounded-xl` (12px), `rounded-2xl` (16px), `rounded-3xl` (24px), `rounded-full`

Borders

Light: `border-slate-200`; Dark: `dark:border-slate-700|800`; Active: `border-brand-500/40+`

Elevation

`shadow-sm` default, `shadow-md` interactive, `shadow-xl|2xl` for modals/overlays

Motion, Opacity, Layering

Duration / Easing

`duration-200|300`, `ease-in-out`, `transition-colors|all`

Interaction

`active:scale-95`, hover opacity, shimmer + pulse for special states

Z-Index

Header `z-30`; Sticky bars `z-20/30`; Modal overlays `z-50+`

Token Governance Rules

  • 1. Use `brand-*` for primary actions, emphasis, and links — avoid ad-hoc blue tokens.
  • 2. Use `slate-*` for surfaces, typography, separators, and dark-mode backgrounds.
  • 3. Keep custom hex usage only for logo assets, illustrations, or chart palettes.
  • 4. New components should inherit these token groups before adding new values.
